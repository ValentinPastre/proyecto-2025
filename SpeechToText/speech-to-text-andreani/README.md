# STT-Argentino-RNN y Whisper-Fine-Tuned-Argentino: Reconocimiento de Voz en Español Argentino

Esta carpeta contiene dos implementaciones de modelos de Speech-to-Text (STT) optimizados para el español Argentino:
1. **Modelo BiLSTM (RNN):** Un modelo liviano entrenado desde cero.
2. **Modelo Whisper (Transformer):** Un modelo fine-tuneado de OpenAI con alta precisión.

Ambos scripts incluyen inferencia en tiempo real mediante micrófono con modalidad push to talk.

## Instalación:
crear un virtual enviroment, luego dentro del envirement:
pip install -r requirements.txt
